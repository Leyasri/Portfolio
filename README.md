\# Personal Portfolio Website



This is a responsive personal portfolio website built using HTML, CSS, and

vanilla JavaScript. The website presents my academic background, projects,

technical skills, awards, and contact information in a clean and interactive

layout.



---



Features



\- Responsive single-page design

\- Dark / Light mode toggle using JavaScript

\- Smooth scrolling navigation

\- Mobile-friendly hamburger menu

\- Separate CSS and JavaScript files

\- No external JavaScript libraries used



---



Technologies Used



\- HTML5

\- CSS3

\- JavaScript (ES6)

\- Git and GitHub
